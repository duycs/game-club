## GAME CLUB APP

### Environments
- Install node v18.15.0, avaliable with node version > 14
- Install angular 17.3: ```npm install -g @angular/cli```
- Default local environment in development

### Build
- ```npm install```
- ```ng build```

### Run
- ```ng serve```
- The App in local development run at: ```https://localhost:4200```
